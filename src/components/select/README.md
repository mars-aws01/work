# nk-select

**Inputs**

| 属性 | 说明 |  类型 | 默认值 |
| --- | --- | --- | --- | --- |
| placeholder | 占位提示文字 | string | '' |
| allowClear | 是否显示clear按钮 | boolean | false |
| ngModel | 选中的值 | any | - |


# nk-option

**Inputs**

| 属性 | 说明 |  类型 | 默认值 |
| --- | --- | --- | --- | --- |
| label | 设置显示文本 | string | - |
| value | 设置选择之后的值 | any | - |
| disabled | 设置是否为禁用状态 | boolean | false |
