## nk-image-zoom

**Slots（ng-content占位符）**

| Name | Description |
| --- | --- |
| '' | 放置img标签内容 |

**Events（Output）**

| Name | Paramters | Description |
| --- | --- | --- |
| change | open: boolean | 当图片状态变化时触发，传入当前是展开还是收缩 |

**Usage**
```html
<nk-image-zoom>
  <img src="http://10.16.75.10:8001/img/demo/m2.jpg" alt="" style="width: 100px;height: 100px;">
</nk-image-zoom>
```
